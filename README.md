### Installation
After you clone the repository
> Import database.sql to your postgres DB
> Edit the database config in ormconfig.json:
```typescript
// ./ormconfig.json
{
    "type": "postgres",
    "host": "localhost",
    "port": 5432,
    "username": "postgres",
    "password": "",
    "database": "quizApp",
    "entities": ["dist/**/*.entity.js"],
    "synchronize": true
  }
```
> npm install
> npm run start
> swagger : localhost:3000/api
> url : localhost:3000/api/v1
